# EduPortal — Сайт колледжа на Django

## Быстрый старт

```bash
# 1. Установить Django
pip install django

# 2. Применить миграции (уже применены, db.sqlite3 включён)
python manage.py migrate

# 3. Запустить сервер
python manage.py runserver
```

## Доступные страницы

| URL | Описание |
|-----|----------|
| http://127.0.0.1:8000/hello/ | Урок 1 — Hello страница |
| http://127.0.0.1:8000/students/ | Список студентов (CRUD) |
| http://127.0.0.1:8000/students/add/ | Добавить студента |
| http://127.0.0.1:8000/groups/ | Группы (ForeignKey) |
| http://127.0.0.1:8000/clubs/ | Кружки (ManyToMany) |
| http://127.0.0.1:8000/login/ | Вход |
| http://127.0.0.1:8000/register/ | Регистрация |
| http://127.0.0.1:8000/admin/ | Админ-панель |

## Данные для входа

- **Суперпользователь:** admin / admin123
- **Тестовый пользователь:** зарегистрируйте через /register/
